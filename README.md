# Bellestride - Premium Shoes E-commerce Platform

A mobile-friendly web application for selling shoes with cash on delivery payment system.

## Features

### Customer Features
- Browse shoe collection with category filtering (Men, Women, Sports)
- View detailed product information
- Add items to shopping cart
- Adjust quantities in cart
- Checkout with cash on delivery
- Order confirmation system
- Responsive mobile design

### Admin Features
- Password-protected admin panel (password: `Jillian`)
- Add new products with images
- Delete products
- View recent orders
- Manage inventory

### Technical Features
- Local storage for data persistence
- Mobile-responsive design
- Modern UI with Tailwind CSS
- Smooth animations and transitions
- No login required for customers
- Shareable URL for easy distribution

## Usage

### For Customers
1. Open the web app using the provided URL
2. Browse available shoes
3. Click on products to view details
4. Add items to cart
5. Proceed to checkout
6. Fill in delivery information
7. Confirm order (cash on delivery)

### For Admin
1. Click the shield icon in the header
2. Enter admin password: `Jillian`
3. Add new products using the form
4. Manage existing products
5. View customer orders
